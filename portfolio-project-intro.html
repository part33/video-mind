Video Mind AI

AI 视频解析与创作工作台

面向短视频创作者的 AI 视频解析与创作工作台，支持视频导入、内容摘要、场景拆解、脚本改写与提示词生成。 项目重点解决大文件上传、AI 分析耗时、重复分析和结果复用问题，将参考视频整理为可继续编辑的创作结果。

Video Upload Scene Breakdown Script Rewrite Prompt Bundle
INPUT

视频素材

支持本地视频导入，将原始素材进入统一分析链路。

PROCESS

结构化分析

围绕字幕、摘要、场景、动作和镜头信息进行拆解。

OUTPUT

创作结果

输出可查看、可复用、可继续编辑的脚本与分镜内容。

查看应用演示 查看 GitHub
CORE WORKFLOW
Upload -> Analyze -> Rewrite

从视频导入、异步分析到结果查看与脚本改写，形成完整创作闭环。

ENGINEERING FOCUS
大文件上传 / 异步任务 / 去重复用

围绕真实视频处理场景设计分片上传、MD5 指纹、异步分析与并发控制。

TECH STACK
Spring Boot + Vue 3

结合 MySQL、Redis、Redisson、RocketMQ、MinIO 与大模型接口完成全栈实现。

HIGHLIGHTS

技术亮点

UPLOAD

分片上传与文件指纹

实现分片上传、文件合并与服务端 MD5 计算，支持重复内容识别与分析结果复用。

CONCURRENCY

Redisson 锁与限流

通过分布式锁控制重复提交，结合限流思路降低高频 AI 请求带来的算力浪费。

ASYNC

RocketMQ 异步链路

将字幕转写、摘要生成、场景拆解等耗时任务与上传请求解耦，提升扩展性。

DATA

AI 结果结构化

用 MySQL 维护视频项目、场景分段和改写草稿，让模型输出可查看、可复用。
