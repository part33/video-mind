# Video Mind AI Client

This directory contains the frontend application for Video Mind AI.

## Stack

- Vue 3
- Vite
- GSAP

## Local Development

```bat
npm install
npm run dev -- --host 127.0.0.1 --port 5173
```

## Build

```bat
npm run build
```

## Frontend Scope

The current client includes:

- creator workspace home
- upload and link import entry
- project rail and project detail view
- transcript, summary, and scene breakdown display
- rewrite draft and prompt bundle preview
